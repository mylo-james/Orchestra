# Story 1.3: Integrate Tasks, Templates, Checklists as Orchestra Resources

## Status
Spec In Progress

## Story
**As a** Scrum Master,
**I want** BMad tasks, templates, and checklists integrated as Orchestra resources,
**so that** the system can execute them via a ResourceLoader and engines without code changes.

## Acceptance Criteria
1. Define resource schemas for tasks, templates, checklists
2. Implement ResourceLoader, TaskEngine, TemplateProcessor, ChecklistEngine interfaces
3. Validate resources via JSON Schemas with CI checks
4. Provenance, signing, and trust levels captured on outputs
5. Hot-reload resource updates atomically with versioned cache keys

## Tasks / Subtasks
- [ ] Design resource schemas and metadata (AC: 1)
- [ ] Implement loader/engines scaffolding (AC: 2)
  - [ ] `orchestra/system/resource_loader.py`

- [ ] Attach provenance/signing/trust levels to outputs (AC: 4)
- [ ] Implement atomic hot-reload and versioned caching (AC: 5)
