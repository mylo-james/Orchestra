# Story 1.6: Directory Scaffolding for teams/ and projects/ Overlays

## Status
Draft

## Story
**As a** Scrum Master,
**I want** standard directory scaffolding for team and project overlays,
**so that** configurations and resources can be organized and loaded correctly.

## Acceptance Criteria
1. Define directory layout for `teams/` and `projects/`
2. Provide example overlay files and README
3. Loader resolves overlays by (team_id, project_id)
4. CI checks ensure directory correctness
5. Hot-reload detects overlay changes

## Tasks / Subtasks
- [ ] Propose directory structure with examples (AC: 1,2)
- [ ] Implement loader resolution order (AC: 3)
- [ ] CI checks for presence and schema validity (AC: 4)
- [ ] Hot-reload tests (AC: 5)
