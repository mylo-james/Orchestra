## Status

## Story
**As a** Scrum Master,
**I want** tag-based targeting for personas and resources,
**so that** broadcast and policy cascades can target subsets efficiently.

## Acceptance Criteria
1. Tag schema and matching rules defined
2. Registry supports tag queries and selection
3. CLI/UI supports tag-based filters
4. Tracing/logging includes tags for executions
5. Unit and integration tests cover matching and filters

## Tasks / Subtasks
- [ ] Define tag model and query semantics (AC: 1)
- [ ] Implement registry support and APIs (AC: 2)
- [ ] CLI filters for listing/targeting (AC: 3)
- [ ] Add tags to trace/log contexts (AC: 4)
- [ ] Tests for matching, conflicts, and edge cases (AC: 5)
